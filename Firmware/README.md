# Firmware Directory

This directory contains the source firmware code for the HexadecipadV2.

## Files

- **main.py** - Main entry point for the CircuitPython firmware
- **hexadecipad_control.py** - Control logic and functionality for the Hexadecipad (key handling, LED control, etc.)

## Hardware Requirements

This firmware is designed for:
- Seeed Studio XIAO RP2040 microcontroller
- CircuitPython 10.0.3 or compatible version


For production-ready firmware files, see the `/Production/Firmware/` directory.
