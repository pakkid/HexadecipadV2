# Other Files Directory

This directory contains supporting files including 3D models and documentation images.

## Subdirectories

### 3d models/
Contains 3D STEP models of components used in the design:
- **asm_mx_asm_PCB.stp** - MX mechanical switch assembly with PCB
- **Encoder.STEP** - Rotary encoder 3D model
- **Seeed Studio XIAO RP2040 v26.step** - 3D model of the XIAO RP2040 microcontroller

### images/
Contains project documentation images:
- **case with pcb.png** - Render showing the PCB mounted in the case
- **pcb filled.png** - PCB view with filled copper zones visible
- **pcb unfilled.png** - PCB view without filled zones
- **schematic.png** - Electronic schematic diagram

## Usage

These files are primarily used for:
- 3D models: CAD visualization and mechanical clearance verification
- Images: README documentation and project showcase
