# Images

This directory contains documentation and visualization images for the HexadecipadV2 project.

## Files

- **case with pcb.png** - 3D render showing the PCB installed in the case assembly
- **pcb filled.png** - PCB layout view with filled copper zones (ground and power planes visible)
- **pcb unfilled.png** - PCB layout view with unfilled zones (traces and routing more visible)
- **schematic.png** - Complete electronic schematic diagram of the circuit

## Usage

These images are used in:
- Project README and documentation
- Assembly guides
- Design reviews
- Showcasing the project
